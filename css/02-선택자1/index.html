선택자의 조합

  • 클래스나 아이디를 호출시 여백을 이용해서 다수의 클래스명을 연결, id는 1개만 지정가능
  • combine(연결)선택자 : 여러 선택자를 연결하는 방식, 타입명은 앞쪽에서 사용
  • combination선택자(여러가지방법)
  • comma combination쉼표조합선택자 : 쉼표를 이용해서 다수의 선택자를 연결
  • 자손(decendent)선택자 : 여백을 이용한 선택자
  • 자식(children)선택자를 이용한 >선택자
  • 전체(universal)선택자 : * (전부다)
