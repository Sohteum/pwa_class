dl (definition list)

용어나 의미를 정의하는 목록

웹언어

html
웹 문서의 시멘틱 구조를 설계하는 언어
css
웹 문서의 레이아웃 및 시작적 구조를 설계하는 용어

목록 요소의 특징

  • ul(li만 자식으로 허락됨), type속성(disc, circle, square)
  • ol(li만 자식으로 허락됨), type속성(1,A,a,I,i)
  • dl(dt,dd만 자식으로 허락됨)
  • type속성 대신 list-style과 가상 엘리먼트를 이용해 불릿기호를 대체함

불릿기호 출력하기

  • 아바타
  • 슬램덩크
  • 영웅
