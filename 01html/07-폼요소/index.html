form요소

  • form요소는 데이터 전송범위를 의미
  • fieldset: 폼요소 내에서 데이터의 범위를 구분
  • legend: fieldset의 제목
  • label : 입력요소의 입력값의 참조할 내용, for(레이블에)와 id속성(입력요소)을 사용(인풋하나당 무조건레이블하나권장)
  • placeholder속성 : 글자나 숫자를 입력하는 폼요소에 적용가능
  • size속성 : text,password의 넓이를 설정하는 속성(가급적사용하지말것.크로스브라우징 준수하지못함)
  • maxlength속성 : text,password의 글자길이 제한
  • autocomplete속성 : 자동완성기능, name속성을 가져야함
  • option요소의 label속성은 텍스트를 대체하는 기능을 함
  • button type : submit,reset,button
