폼요소

  • form요소는 데이터 전송범위를 의미
  • fieldset : 폼요소내에서 데이터의 범위를 구분
  • legend : fieldset의 제목
  • label : 폼 요소기때문에 인라인! 입력요소에 입력값의 참조할 내용, for와 id속성을 사용
  • placeholder속성 : 글자나 숫자를 입력하는 폼요소에 적용가능
  • size속성 : text,password, email의 넓이를 설정하는 속성(크로싱브라우저를 준수하지 않는다)
  • maxlength속성 : text, password, email의 글자길이 제한
  • button type = submit, reset, button
  • option요소의 label속성은 텍스트를 대체하는 기능을 한다.
  • autocomplete속성 : 자동완성기능, name속성을 지정해야 함
  • file요소 : multiple(다수의 파일을 선택 할 수 있는 기능), accept(파일 유형을 선택할 수 있는 기능)
  • email : 유효성 검사(제대로 입력했나/틀렸나 맞나) 체크기능제공, mobile 사용시 키패드 ui환경을 제공(키패드가 알아서 튀어나오는)
  • date, time : 날짜와 시간을 입력
  • number, range : 숫자만 입력가능, min, max, step 속성이 허락됨
