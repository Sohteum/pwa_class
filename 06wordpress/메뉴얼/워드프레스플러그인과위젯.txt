■ 플러그인 설치방법
  ● 대시보드/플러그인/새로추가
  ● 검색설치 혹은 플러그인파일(zip파일)을 업로드
  ● 설치 후 활성화
  ● 설치된 플러그인 메뉴는 대시보드메뉴에, 설정, 도구메뉴에 등록된다

■ 플러그인 출력방법
  ● shortcode를 관리자화면에서 추가, 코드편집기를 사용하길 권장
    예시 : [shortcode]
  ● 템플릿을 이용해 출력
    예시 : <?php echo do_shortcode('[shortcode]');?> 
    
■ Remove Category URL 
  ● url의 category출력을 제거
  ● 별도의 설정이 필요하지 않음

■ Breadcrumb NavXT
  ● 웹사이트 Breadcrumb 출력
  ● 대시보드/설정 안으로 메뉴가 등록됨
  ● 사용법
    -출력함수
    <?php if(function_exists('bcn_display')){ bcn_display(); }?>
  ● 세부옵션
    - General / Breadcrumb Separator : 구분기호
    - General / Home Template , Home Template (Unlinked) : 홈출력설정
    - 페이지나 글 출력방식 설정
      · Post Types / 글 template 안쪽의 내용은 여백 한칸 추가
      · Post Types / 글 template 또는 페이지 template 태그를 수정
      · Texonomies / 카테고리 template 태그를 수정

■ kboard
  ● 게시판 플러그인
  ● 대시보드/kboard메뉴
    ○ 게시판생성
      - 기본설정  
        · 출력될 페이지 선택
        · 스킨
        · 최근게시물 코드
          <?php 
            $homeURL = home_url(bloginfo('url'));
            $kboardID = "1" ;		
            $kboardPAGE = "notice";
            $kboardCnt = "3";
            $kboardSTR = '[kboard_latest id="1" url="'.$homeURL.$kboardPAGE.'" rpp="'.$kboardCnt.'"]'; 
            echo do_shortcode( $kboardSTR ); 
          ?>
      - 권한설정
        · 읽기, 쓰기 권한 
        · 제한없음, 로그인사용자, 직접선택(관리자선택가능)
    - 게시판목록 및 관리
    - contact-form 스킨 사용시 기본설정 메뉴/관리자 페이지에서 게시판보기/활성화
    - 고급설정/아이프레임으로보기/활성화

  ● 최신글
    ○ 게시판설정
      대시보드 / kboard / 게시판목록및관리 / 기본설정 / 최신글이동페이지 연동

    ○ 출력코드
      <?php 
      $homeURL = home_url( '/' );
      $kboardID = "1" ; //게시판id		
      $kboardPAGE = "notice"; //페이지 슬러그
      $kboardCnt = "5";
      $kboardSTR = '[kboard_latest id="1" url="'.$homeURL.$kboardPAGE.'" rpp="'.$kboardCnt.'"]'; 
      echo do_shortcode( $kboardSTR ); 
      ?>


■ Meta Box (Metabox.io)
  ● 글의 메타정보를 추가로 설정할 수 있는 플러그인
  ● functions.php
  add_filter( 'rwmb_meta_boxes', 'your_prefix_meta_boxes' );
  function your_prefix_meta_boxes( $meta_boxes ) {
    $meta_boxes[] = array(
      'title'      => '상품세부정보',
      'post_types' => 'post',
      'fields'     => array(
        array(
          'id'   => 'producer',
          'name' => '제작사',
          'type' => 'text',
        ),
        array(
          'id'   => 'model-id',
          'name' => '모델명',
          'type' => 'text',
        ),  
        array(
          'id'   => 'date',
          'name' => '제작일자',
          'type' => 'date',
        ),
        array(
          'id'   => 'price',
          'name' => '가격',
          'type' => 'number',
        ),
        array(
          'id'   => 'desc',
          'name' => '상품설명',
          'type' => 'text',
        ),
      ),
    );
    return $meta_boxes;
  }
  ● 출력함수
  <?php echo rwmb_meta( '출력하고 싶은 목록의 아이디' );  ?>
