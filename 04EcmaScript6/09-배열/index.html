배열

  • 배열의 선언방식과 호출 
            let arr = ['가', '나', '다']
        arr[0] //'가'
  • array.length : 배열의 길이(1부터 시작, index는 0부터 읽음) 
            let arr = ['가', '나', '다']
        arr.length //3
  • 배열의 값 추가와 삭제 
            push(배열의 뒷쪽에 값을 추가)
        unshift(배열의 앞쪽에 값을 추가)
        pop(뒷쪽제거)
        shift(맨앞제거)
        arr.push('라') = ['가', '나', '다', '라']
        arr.pop() = ['A','가', '나', '다'] 
        arr.shift() = ['가', '나', '다'] 
        인덱스는 0부터, 배열은 1부터
  • 배열의 특정 인덱스를 기준으로 값 추가, 삭제 
            let arr = [0, 1,2,3]
        arr.splice(2,0,'가')//[0,1,'가', 2 ,3]
        arr.splice(2,2)//[0, 1 ,3]
  • split : 문자열을 배열로 변환 
            let str = '안녕하세요'
        str.split() //['안녕하세요'] 
        str.split('') //['안','녕','하','세','요'] 
        let str = '010-1234-4567'
        str.split('-') // ['010', '1234','4567']
  • join : 배열을 스트링(문자열)으로 변환 
            let arr = ['홍', '길', '동']
        arr.join('-') '홍-길-동'
        arr.join('') '홍길동'
        arr.join() '홍,길,동'

1. push, unshift, pop, shift ( 배열의 값 추가와 삭제)

2. splice 배열의 특정 인덱스를 기준으로 추가나 삭제

3. split(문자열을 배열로 전환)
