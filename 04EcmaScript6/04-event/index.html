• event.currentTarget: 이벤트 바인딩 된 엘리먼트를 호출
    event.target: nested구조로 된 엘리먼트일경우 이벤트 위치에 존재하는 최 하위 요소 호출(맨 안에 있는 자식)
  • nested구조에서만 일어나는 일인거지.
    capturing : 이벤트를 감지하는 순서(부모먼저)
    bubbling : 이벤트를 실행하는 순서(자식먼저)
  • e.preventDefault: 브라우저에서 설정된 엘리먼트의 기본 행동을 막는다.
    e.stopPropagation :버블링을 중단한다(부모의 행동을 막는다)
