float을 이용한 그리드 레이아웃

다양한 레이아웃 구현 목적(주로 높이가 다른격자/그리드는 높이랑 너비까지 다른격자)

  • float이 적용된 자식객체의 높이를 부모객체가 인식하지 못함.영향을 못줌 clearfix를 이용해 해결가능 
              부모요소::after {
            content:"";
            display:block;
            clear:both;
          }
    마지막 요소에 클리어보스를 적용하면 되는데 넣을 수 있는 타입이 딱이 없어서 가상요소를 만들어 클리어보스적용
  • float이 적용되면 display 속성을 잃는다 margin의 auto를 사용할 수 없음 inline요소도 크기를 지정할 수 있음
  • float이 적용된 다음요소는(인접형제 adjacent siblings)float과 clear중 하나를 선택해야함(아니면없어져버림)
    -float:앞쪽 형제들과 나란히 서겠다
    -clear:형제들과 관계를 끊겠다.(오 왼 둘함)
  • float레이아웃시 margin-bottom 사용 권장
    float이 적용되지 않은 객체는 위쪽에 위치하는 float요소와 margin이 적용되지 않음
