grid system

  • display속성이 grid인 컨테이너는 block레벨과 유사함
    grid컨테이너의 자식요소는 기존의 display속성이 사라짐
  • grid컨테이너의 격자 설정
    grid-template-rows: repeat(개수, 한줄의너비(auto,1fr))
    grid-template-columns: repeat(개수, 한칸의너비(auto,1fr))
    gap:그리드간의 여백 설정(컨테이너에서 설정함)
  • grid를 설정
    grid-row: 2/6 (2번째 줄에서 시작해서 6번째 전에서 끝남)
    grid-column: 2/sapn 3 (2번째 칸에서 시작하는 3칸짜리)
    grid-area:2/3/7/4; (2번째에서 시작 6번째 줄에서 끝남, 3번째 칸에서 시작, 3번째칸에서 끝)
