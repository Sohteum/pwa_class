css Cascading style sheet

우선순위와 상속이 존재하는 디자인 양식

type선택자(태그선택자)

사용법

태그명{속성:값} 형태로 선언

특징

  • 동일한 태그에 모두 적용
    • 자식태그선택자가 우선순위가 높다 agent스타일(브라우저기본스타일)과 겹칠(충돌날수있음) 수 있다

class 선택자

사용법

.클래스명{속성:속성값;} 을 이용해서 선언 후 class속성을 이용해서 호출

특징

  • 타입선택자보다 우선순위가 높다
  • 반복호출이 가능하다
  • 수정관리가 용이

id선택자

사용법

#아이디명{속성:속성값} 선언 후, id속성을 이용해서 호출

특징

  • class선택자보다 우선순위가 높음
  • 반복호출이 허락되지 않으며, 수정관리가 어려움
  • html문법이나 프로그램용도로 사용

inline 스타일

사용법

요소의 style속성을 지정 후 css를 작성

특징

  • 우선순위가 id보다 높음
  • 스크립트를 이용해 css를 제어하는데 사용

css의 특징

  • 요소단위로css가 적용됨
  • 우선순위 : 인라인스타일, id, class, 자식태그, 부모태그
  • 글자정렬, 글자관련 스타일은 상속이 됨.
