flex

자식요소의 레이아웃을 설계하는 목적 flex요소의 자식요소는 display속성이 유효하지 않음

flex의 종류

  • display : flex

    • (flex를 적용한)부모요소는 block레벨과 동일한 성격을 가짐
    • 자식들의 크기가 부모의 크기와 동일해짐

  • display : inline-flex

    • 부모의 크기가 자식의 크기에 의해 설정됨(크기를 안주는게 목적인거
    • 부모의 크기를 지정하지 않음
    • 자식의 크기를 %로 지정하지 않는다(부모크기없으니 적용안됨)
    • 수평레이아웃에 사용함

  • column, column-reverse (자식요소들의 너비가 동일해짐)

flex속성

부모요소가 flex속성일경우 자식요소에게 넓이를 설정하는 속성

  • 크기를 자동으로 설정
  • 축이row일경우 너비설정, 축이col일경우 높이설정
  • flex속성의 값이 설정되면 크기를 지정할 수 없음
  • flex계산요령
    자식요소가 3개일 경우, 1,1,2를 지정하면 (2/(1+1+2))*100 이렇게 계산할일은 별로 없는데 -다른애들보다 몇배. 이때는 플렉스가 쉬움(자식기준으로 줄때) -퍼센트가 쉬울때도 있고요(부모기준으로 줄때)
